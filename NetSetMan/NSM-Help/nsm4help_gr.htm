Περιγραφή

Το NetSetMan είναι ένας διαχειριστής ρυθμίσεων δικτύου που μπορεί εύκολα να κάνει εναλλαγή μεταξύ 6 διαφορετικών εικονικών προφίλ συμπεριλαμβάνοντας:

  • Διευθύνσεις IP (IPv4 και IPv6!)
  • Πύλες Εξόδου (συμπεριλαμβάνει Metric)
  • DNS Εξυπηρετητές
  • WINS Εξυπηρετητές
  • Διαχείριση WiFi
  • Όνομα Η/Υ
  • Ομάδα εργασίας
  • Προκαθορισμένος Εκτυπωτής
  • Δίσκοι Δικτύου
  • Κατάσταση Προσαρμογέων Δικτύου (NIC)
  • Συνδέσεις (RAS / DUN / PPP / VPN)
  • Επίθημα DNS
  • Route Πίνακας
  • SMTP Εξυπηρετητής
  • Εγγραφές αρχείου Hosts
  • Ρυθμίσεις συστήματος
  • MAC Διευθύνσεις
  • Σενάρια (BAT, VS, JS, ...)
  • Εκτέλεση προγράμματος
Επιπρόσθετα, μόνο στην έκδοση Pro:
  • Μπορεί να χρησιμοποιηθεί στην εργασία
  • Απεριόριστα προφίλ
  • Δίκτυο-Τομέας
  • Πλήρης Ρυθμίσεις Διακομιστή Μεσολάβησης &
    Αρχική Σελίδα Περιηγητή
    (Internet Explorer, Firefox, Opera)

Το NetSetMan σας προσφέρει ότι σας έλειπε στα Windows μέχρι τώρα:
Ένα ισχυρό, εύκολο στη χρήση interface για να διαχειριστείτε όλες τις ρυθμίσεις του δικτύου σας με μια ματιά.
Δοκιμάστε το και ποτέ δεν θα θέλετε να είστε χωρίς αυτό ξανά.

Όλες οι ρυθμίσεις και αλλαγές, αποθηκεύονται αυτόματα σε ένα ξεχωριστό αρχείο (settings.ini) έτσι ώστε ενέργειες όπως δημιουργία αντιγράφου ασφαλείας, μεταφοράς ή αναβάθμισης σε άλλη έκδοση να πραγματοποιούνται χωρίς κανένα πρόβλημα απολύτως.

Ενεργοποιήστε ένα αποθηκευμένο προφίλ με ένα κλικ ή ακόμα και εντελώς αυτόματα!

Βασικές Πληροφορίες
  • Προφίλ
    Κάθε προφίλ αντιπροσωπεύει ένα σύνολο ρυθμίσεων που απαιτούνται για μια συγκεκριμένη τοποθεσία ή σενάριο. Με τη χρήση υπο-προφίλ, πολλοί προσαρμογείς δικτύου μπορούν να ρυθμιστούν με ένα προφίλ. Μετονομάστε ένα προφίλ με διπλό κλικ στην καρτέλα λεζάντα της.
  • Χειροκίνητη ρύθμιση
    Θα πρέπει να ρυθμίσετε χειροκίνητα τα απαιτούμενα προφίλ. Το να μαντέψει κανείς τις σωστές ρθυμίσεις για μια άγνωστη τοποθεσία είναι αδύνατο. Ο κύριος σκοπός της εφαρμογής είναι ο αυτοματισμός των τακτικών αλλαγών των ρυθμίσεων που συνήθως κάνετε χειροκίνητα.
  • Επιλογή των ρυθμίσεων
    Διαβάζοντας όλες τις τρέχουσες ρυθμίσεις ενός προφίλ μπορεί να σας δώσει ένα σημείο εκκίνησης, αλλά πρέπει ακόμα να αποφασίσετε ποιες ρυθμίσεις θα εφαρμοστούν κατά την ενεργοποίηση αυτού του προφίλ. Συνιστάται έντονα να περιορίσετε τις ρυθμίσεις σε εκείνες που πραγματικά έχουν μεταβληθεί. Για παράδειγμα, δεν είναι απαραίτητο να εφαρμοστεί η ρύθμιση που αφορά το όνομα του υπολογιστή, εάν αυτό δεν πρόκειται να αλλάξει ποτέ.
  • Διαμόρφωση των ρυθμίσεων
    Για να διαμορφώσετε μια ρύθμιση επιλέξτε την και στη συνέχεια είτε εισάγετε την επιθυμητή τιμή απευθείας είτε ανοίξετε το παράθυρο διαλόγου ρυθμίσεων για τη συγκεκριμένη ρύθμιση.
  • Ενεργοποίηση Προφίλ
    Από προεπιλογή κανένα προφίλ δεν ενεργοποιείται αυτόματα, δηλαδή καμία ρύθμιση του συστήματος σας δεν μεταβάλλεται χωρίς την συναίνεση σας. Τούτου λεχθέντος, μπορείτε να χρησιμοποιήσετε την AutoSwitch δυνατότητα της εφαρμογής ώστε να ενεργοποιήσετε αυτόματα συγκεκριμένα προφίλ βάση ελεύθερα προσαρμόσιμων συνθηκών.
  • Πρόσβαση στο εικονίδιο ειδοποιήσεων
    Συνιστάται να διατηρείτε το NetSetMan ελαχιστοποιημένο στην περιοχή ειδοποιήσεων εξασφαλίζοντας ότι το εικονίδιο θα είναι ορατό πάντα. Με τον τρόπο αυτό, μπορείτε να ενεργοποιήσετε γρήγορα ένα προφίλ κάνοντας κλικ στο εικονίδιο και επιλέγοντας ένα προφίλ. Επιπλέον, θα έχετε μια χρήσιμη εικόνα για τις τρέχουσες διευθύνσεις IP της σύνδεσής σας με ένα απλό πέρασμα του ποντικιού επάνω από αυτό.
Προφίλ

Οι καρτέλες αντιπροσωπεύουν τα διαθέσιμα προφίλ. Μπορείτε να αποθηκεύσετε 6 διαφορετικά σετ ρυθμίσεων σε αυτές τις καρτέλες.

Αν θέλετε να αλλάξετε τις ρυθμίσεις δύο διαφορετικοών προσαρμογέων δικτύου όταν ενεργοποιείτε ένα προφίλ, μπορείτε να χρησιμοποιήσετε τα Υπο-προφίλ. Κάντε κλικ στο "+" - κουμπί δίπλα από την αναπτυσσόμενη λίστα και μια νέα γραμμή μενού θα εμφανιστεί που περιέχει δύο υπο-προφίλ (το τρέχον και ένα νέο κενό). Η λεζάντα θα είναι αυτόματα το όνομα του προσαρμογέα δικτύου που έχει επιλεγεί σε αυτό το υποπροφίλ και δεν μπορεί να αλλάξει. Μπορείτε να δημιουργήσετε όσα υπο-προφίλ θέλετε. Όλα αυτά θα ενεργοποιηθούν το ένα μετά το άλλο κατά την ενεργοποίηση του προφίλ. Μπορείτε να αλλάξετε τη σειρά με σύρσιμο & απόθεση.

Όλες οι αλλαγές αποθηκεύονται αυτόματα έτσι ώστε να μη χάνεται τίποτα κατά λάθος.

Μπορείτε να μετονομάσετε ένα προφίλ με το δεξί κλικ και επιλέγοντας "Μετονομασία", ή απλά κάντε διπλό κλικ στην λεζάντα της καρτέλας του.

Για να διαχειριστείτε όλα τα προφίλ σας πιο εύκολα, μπορείτε να χρησιμοποιήσετε τον Διαχειριστή Προφίλ που θα βρείτε στο μενού. Επιτρέπει την επεξεργασία προφίλ, υποπροφίλ και ομάδων (μόνο στο NetSetMan Pro) με το ποντίκι σας ή κάποιες κοινές συντομεύσεις πληκτρολογίου (πλήκτρα βέλους για πλοήγηση, F2 για μετονομασία, κλπ). Κατά τη χρήση συρσίματος & απόθεσης για να αλλάξετε τα στοιχεία σας θα παρατηρήσετε κάποιους αναπόφευκτους περιορισμούς ανάλογα με το αντικείμενο, λόγω δεδομένων τεχνικών συνθηκών: Τα προφίλ δεν μπορούν να διαγραφούν, Πολλαπλά Προφίλ δεν μπορούν να μετονομαστούν ενώ τα προφίλ μπορούν να ανταλλαχθούν, αλλά δεν μετακινούνται. Τα πλαίσια επιλογής υποδεικνύουν εάν το προφίλ θα είναι ορατό στην περιοχή ειδοποιήσεων αλλά και στην Συμπαγή κατάσταση.

Προσαρμογείς δικτύου (NIC)

Όλοι οι διαθέσιμοι προσαρμογείς δικτύου εμφανίζονται με το όνομά και τον τύπο τους. Επιλέξτε τον προσαρμογέα δικτύου που θα θέλατε να διαχειριστεί το τρέχον (υπο-) προφίλ.

Εάν κάποια NIC δεν είναι διαθέσιμη στη λίστα (π.χ. επειδή είναι απενεργοποιημένη ή εικονική) μπορείτε να διαχειριστείτε τη λίστα κάνοντας κλικ στο κουμπί Ανανέωση και είτε να επικαιροποιήσετε την λίστα γρήγορα είτε να επιλέξετε την Διαχείριση προσαρμογέων δικτύου. Εκεί θα βρείτε μια πλήρη λίστα με όλες τους διαθέσιμες προσαρμογείς δικτύου. Ελέγξτε όλους τους προσαρμογείς δικτύου που θέλετε να έχετε στη λίστα σας.

Σημείωση: Όλοι οι προσαρμογείς δικτύου που περιλαμβάνονται στον εν λόγω κατάλογο είναι επίσης ορατοί στην περιοχή ειδοποιήσεων. Αναιρέστε την επιλογή προσαρμογέων δικτύου που δεν θέλετε ώστε να μην εμφανίζονται πλέον ούτε στην λίστα αλλά ούτε και στην περιοχή ειδοποιήσεων

Εάν αντιγράψετε το αρχείο ρυθμίσεων (settings.ini) σε ένα άλλο σύστημα, το NetSetMan θα προσπαθήσει να επαναθέσει εκ νέου τους προσαρμογείς δικτύου στα προφίλ αυτόματα. Ωστόσο, αυτό θα λειτουργήσει μόνο αν το νέο σύστημα έχει έναν προσαρμογέα δικτύου με το ίδιο ακριβώς όνομα (στην πρώτη εικόνα είναι "LAN").

Λειτουργίες Ρυθμίσεων

Ανάκτηση Τρέχοντων Ρυθμίσεων

Για να φορτώσετε μια τρέχουσα ρύθμιση στο προφίλ απλά κάντε κλικ στο σύμβολο δίπλα από το πεδίο εισαγωγής.
Για να πάρετε όλες τις τρέχουσες ρυθμίσεις κάντε δεξί κλικ στο προφίλ και επιλέξτε "Ανάκτηση Τρέχουσας Τιμής".

Εκκαθάριση Ρυθμίσεων

Για να εκκαθαρίσετε τη τιμή μιας συγκεκριμένης ρύθμισης κάντε δεξί κλικ και επιλέξτε "Διαγραφή".

Για να εκκαθαρίσετε την τιμή ενός πεδίου IP απλά κάντε διπλό κλικ στην αντίστοιχη λεζάντα της.

Για να εκκαθαρίσετε όλες τις ρυθμίσεις του τρέχοντα (υπο)προφίλ κάντε δεξί κλικ στην καρτέλα του προφίλ και επιλέξτε "Εκκαθάριση Ρυθμίσεων".

Αντιγραφή & Επικόλληση Ρυθμίσεων

Για να μεταφέρετε τις ρυθμίσεις ανάμεσα σε διαφορετικά προφίλ, μπορείτε να αντιγράψετε και να επικολλήσετε ένα ολόκληρο προφίλ κάνοντας δεξί κλικ στο αρχικό προφίλ, επιλέγοντας "Αντιγραφή Ρυθμίσεων", κάντε δεξί κλικ στο νέο προφίλ και επιλέγοντας "Επικόλληση Ρυθμίσεων".

Για να αντιγράψετε και να επικολλήσετε μόνο μια συγκεκριμένη ρύθμιση απλά κάντε δεξί κλικ και επιλέξτε "Αντιγραφή" ή "Επικόλληση".

Ενεργοποίηση Προφίλ

Για την εφαρμογή όλων των ρυθμίσεων του τρέχοντος προφίλ κάντε κλικ στο "Ενεργοποίηση" ή αν το πρόγραμμα τρέχει ελαχιστοποιημένο επιλέξτε το προφίλ από το μενού στην περιοχή ειδοποιήσεων .
Σημειώστε ότι μόνο ρυθμίσεις που έχουν επιλεγεί θα ενεργοποιηθούν.

Ο διιάλογος Ενεργοποίησης περιέχει πληροφορίες σχετικά με την τρέχουσα πορεία της ενεργοποίησης.
Εδώ μπορείτε να δείτε τις ρυθμίσεις που πρόκειται να εφαρμοστούν και αν η διαδικασία είναι επιτυχής ή όχι. Μπορείτε να ακυρώσετε την διαδικασία ενεργοποίησης κάνοντας κλικ στο κουμπί "Ακύρωση" ανά πάσα στιγμή.

Όλες οι αλλαγές εφαρμόζονται άμεσα. Ωστόσο, σε ορισμένες περιπτώσεις, ορισμένες ρυθμίσεις ενδέχεται να χρειαστούν λίγο χρόνο για να τεθούν σε πλήρη ισχύ ή απαιτούν επανεκκίνηση (ειδικά "Όνομα Η/Υ" και "Ομάδα εργασίας / Τομέας").

Το τμήμα Λεπτομέρειες παρέχει μια επισκόπηση όλων των διαμορφώσεων των ρυθμίσεων για το συγκεκριμένο προφίλ. Για να δείτε περισσότερες πληροφορίες και μηνύματα λάθους στο σύνολο τους, ανοίξτε το τμήμα Καταγραφής.

Για να βρείτε λύσεις για τα πιο συνήθη λάθη και προβλήματα, ανατρέξτε στο FAQ .

Συμπαγής Κατάσταση & Πλήρης Λειτουργία

Πέρα από το κλασικό περιβάλλον "Πλήρης Λειτουργία", όπου μπορείτε να αλλάξετε όλες τις ρυθμίσεις του προφίλ, μια πρόσθετη διεπαφή με την ονομασία "Συμπαγής Κατάσταση" είναι διαθέσιμη. Η εναλλαγή των παραπάνω δυο καταστάσεων είναι εφικτή ανά πάσα στιγμή, κάνοντας κλικ στο αντίστοιχο κουμπί στο κύριο μενού στα δεξιά.

Όταν όλα τα προφίλ έχουν συσταθεί, η Συμπαγής Κατάσταση είναι το ιδανικό περιβάλλον εργασίας. Εδώ έχετε μια συμπαγή επισκόπηση όλων των ρυθμίσεων ενός προφίλ, χωρίς να ανησυχείτε για κατά λάθος τροποποίηση των ρυθμίσεων.

Σε περίπτωση που χρησιμοποιείτε την NSM Διαχείριση (έχοντας την "Αλλαγή ρυθμίσεων προφίλ" μη επιλεγμένη) η Συμπαγής Κατάσταση θα χρησιμοποιηθεί αυτόματα για να αποτρέψει το χρήστη να μεταβάλει τις ρυθμίσεις.

Μενού & Πληροφορίες Χώρου Ειδοποιήσεων

Όταν το NetSetMan εκτελείται θα βρείτε το εικονίδιο του στη γραμμή ειδοποιήσεων (δίπλα στο ρολόι).
Εκεί μπορείτε να ενεργοποιήσετε γρήγορα ένα προφίλ. Απλά κάντε δεξί κλικ στο εικονίδιο για να ανοίξετε το μενού του Χ'ωρου Ειδοποιήσεων και στην συνέχεια να επιλέξετε το προφίλ που θέλετε να ενεργοποιήσετε.

Αν θέλετε να εξαιρέσετε κάποια προφίλ να εμφανίζονται στο χώρο ειδοποιήσεων, μπορείτε να τα αποκρύψετε ξεχωριστά. Για να το κάνετε αυτό, επιλέξτε το προφίλ στο κύριο παράθυρο, κάντε δεξί κλικ στην καρτέλα και κάντε κλικ στην επιλογή "Ενεργό σε λειτουργία Χώρου Ειδοποιήσεων και Συμπαγής Κατάστασης". Το μικρό σύμβολο δίπλα στο κείμενο υποδεικνύει εάν αυτό το προφίλ θα είναι διαθέσιμο στο χώρο ειδοποιήσεων. Η επιλογή έχει επίσης επίπτωση στα προφίλ που εμφανίζονται στην Συμπαγή Κατάσταση. Η ορατότητα των προφίλ μπορεί επίσης να ρυθμιστεί από την Διαχείριση Προφίλ .

Εάν αφήσετε το ποντίκι σας πάνω στο εικονίδιο του NetSetMan στον χώρο ειδοποιήσεων, τότε ο Χώρος Ειδοποιήσεων θα εμφανιστεί. Περιέχει τις τρέχουσες πληροφορίες ΙΡ που χρησιμοποιούνται από το σύστημα. Επειδή τα δεδομένα ανακτώνται σε πραγματικό χρόνο, ενδέχεται να υπάρχει μια μικρή καθυστέρηση. Για να μειώσετε την καθυστέρηση αυτή, πρέπει να απενεργοποίησετε πρόσθετες πληροφορίες, όπως η δημόσια IP και MAC διευθύνσεις στις Προτιμήσεις.

Χαρακτηριστικά
Περιγραφή

Το NetSetMan είναι σχεδιασμένο με τέτοιο τρόπο ώστε να σας επιτρέπει να έχετε μια άμεση επισκόπηση όλων των διαμορφωμένων ρυθμίσεων σε ένα προφίλ. Μερικά από τα πιο σημαντικά χαρακτηριστικά, όπως οι βασικές ρυθμίσεις IP, WiFi και ο προεπιλεγμένος εκτυπωτής μπορούν να ρυθμιστούν άμεσα στην κύρια οθόνη. Άλλες, πιο πολύπλοκες ρυθμίσεις μπορούν να ενεργοποιηθούν μόνο εδώ και θα δείτε μια σύντομη επισκόπηση για την τρέχουσα διαμόρφωση τους. Για να τις τροποποιήσετε, ενεργοποιήστε τις για πρώτη φορά, στη συνέχεια, ανοίξτε το παράθυρο διαλόγου με κλικ στο [...] - κουμπί.

Οι ενότητες που ακολουθούν θα σας δώσουν βασικές οδηγίες και χρήσιμες συμβουλές για το πώς να χρησιμοποιήσετε όλες τις λειτουργίες.

Κατάσταση Προσαρμογέα Δικτύου

Για να ενεργοποιήσετε, να απενεργοποιήσετε ή να επανεκκινήσετε τον επιλεγμένο προσαρμογέα δικτύου χρησιμοποιήστε το κουμπί Κατάσταση Προσαρμογέα Δικτύου. Το προκαθορισμένο χρώμα θα αλλάξει σε άλλο, ανάλογα με την κατάσταση. Επιπρόσθετα, μπορείτε να δείτε την διαμορφωμένη κατάσταση στην σημείωση.

Εάν θέλετε να ενεργοποιήσετε έναν προσαρμογέα και να απενεργοποιήσετε έναν ή περισσότερους ταυτόχρονα, θα πρέπει να δημιουργήσετε υπο-προφίλ για κάθε έναν από τους προσαρμογείς. Μπορείτε να το κάνετε είτε χειροκίνητα είτε με τη χρήση του μενού στο κάτω μέρος του μενού Κατάσταση Προσαρμογέα Δικτύου. Θα δημιουργήσει αυτόματα ένα υποπροφίλ για κάθε προσαρμογέα δικτύου που περιλαμβάνεται στον κατάλογο του προσαρμογέα και θα θέσει την κατάλληλη ρύθμιση ανάλογα. Να θυμάστε ότι μπορείτε να προσαρμόσετε ποιοί προσαρμογείς περιλαμβάνονται στον κατάλογο αυτό με τη χρήση του προσαρμογέα Διαχείρισης Δικτύου .

Προχωρημένες TCP / IP Ρυθμίσεις

Μπορείτε να καθορίσετε πολλαπλές IP, Πύλες, DNS και WINS διευθύνσεις ανά NIC, εφόσον πρέπει να είστε σε πολλά δίκτυα ταυτόχρονα. Για να γίνει αυτό, κάντε κλικ στο κουμπί "IP +" ή επιλέξτε "Προφίλ> Προχωρημένες TCP / IP Ρυθμίσεις" από το κύριο μενού.

Εδώ έχετε επίσης τη δυνατότητα να διαχειριστείτε πιο προηγμένες ρυθμίσεις, όπως πίνακας διαδρομών, επίθημα DNS, IPConfig και NetBIOS.

  • Ενεργοποίηση ομάδας με επιλογή στην λεζάντα
  • Προσθέστε μια νέα τιμή
  • Διαγραφή της επιλεγμένης τιμής
  • Εκκαθάριση όλης της λίστας
  • Ανάκτηση όλων των τρέχοντων τιμών για αυτή τη ρύθμιση από το σύστημα (κάνοντας κλικ στο σύμβολο στο κύριο παράθυρο διαβάζει την πρώτη τιμή μόνο)
  • Αλλάξετε τη σειρά με σύρσιμο και απόθεση
Συνδέσεις

Χρησιμοποιήστε αυτό το χαρακτηριστικό για να συνδέσετε ή να αποσυνδέσετε οποιοδήποτε είδος συνδέσεων των Windows όπως dial-up δίκτυα, οι κινητές ευρυζωνικές ή VPN. Οι συμβατές συνδέσεις εμφανίζονται στα Windows, μαζί με τους προσαρμογείς δικτύου σας, στο παράθυρο διαλόγου Συνδέσεις δικτύου. Από τεχνικής άποψης, όλες οι συνδέσεις που είναι συμβατές με το "RasDial" υποστηρίζονται.

Μπορείτε να επιλέξετε είτε να αποσυνδέσετε όλες τις τρέχουσες εδραιωμένες συνδέσεις ελέγχοντας το πλαίσιο ελέγχου, ή να συνδέσετε / αποσυνδέσετε κάποιες συνδέσεις ξεχωριστά στην σειρά της λίστας.

Εάν η σύνδεση σας δεν εμφανίζεται για κάποιο λόγο, μπορείτε να την προσθέσετε χειροκίνητα επιλέγοντας „Προσθήκη…“ στην λίστα και δίνοντας την ονομασία της όπως αυτή εμφανίζεται στα Windows.

Το NetSetMan υποστηρίζει επίσης διάφορα τρίτα προϊόντα τύπου VPN clients όπως τα OpenVPN, Shrew και Cisco. Εάν ο VPN client σας είναι συμβατός, το NetSetMan θα συμπεριλάβει τα VPN προφίλ του στην λίστα επιλογής.

Hosts

Σε πολύ ειδικές περιπτώσεις μπορεί να είναι αναγκαίο ή χρήσιμο να ορίσετε χειροκίνητα διευθύνσεις IP σε ονόματα ή τομείς. Παρατηρήστε ότι το NetSetMan θα τροποποιήσει μόνο τις δικές του καταχωρήσεις, αφήνοντας τις ήδη υπάρχουσες καταχωρήσεις ανέγγιχτες.

Δίσκοι Δίκτυου

Με το NetSetMan μπορείτε να α(ποα)ντιστοιχίσετε απεριόριστο αριθμό δίσκων δικτύου.
Παρακαλώ διαβάστε τις σημειώσεις οδηγιών στο κάτω μέρος του παραθύρου κάνοντας κλικ στο "[Εμφάνιση Βοήθειας]".

Όταν ένα προφίλ ενεργοποιείται και επιλέγεται η λειτουργία δικτυακού δίσκου, το NetSetMan θα συγκρίνει την λίστα όλων των ήδη υπαρχόντων αντιστοιχίσεων με την λίστα των μονάδων που πρόκειται να αντιστοιχηθούν. Όλες οι αντιστοιχισμένες μονάδες δίσκων που βρίσκονται και στις δύο λίστες (ίδιο γράμμα δίσκου και ίδια διαδρομή) θα παραμείνουν ανέπαφες. Όλες οι άλλες μονάδες του δικτύου θα αποαντιστοιχηθούν πριν την αντοιστοίχηση των νέων.

Σενάρια Εντολών & Προγράμματα

Το NetSetMan σας προσφέρει έναν άνετο τρόπο για να διαχειριστείτε τα αρχεία σεναρίων εντολών σας. Ωστόσο, δεν έχει καμία δική της γλώσσα σεναρίων εντολών. Αυτό σημαίνει ότι μπορείτε να χρησιμοποιήσετε μόνο γλώσσες σεναρίων εντολών που το λειτουργικό σας σύστημα υποστηρίζει. Κοινές γλώσσες σεναρίων εντολών είναι οι VBS, JS είτε ακόμη και η απλή command-line batch files. Χρησιμοποιήστε την αγαπημένη σας μηχανή αναζήτησης για να βρείτε περισσότερες πληροφορίες σχετικά με σενάρια εντολών και τι μπορείτε να κάνετε με αυτά.

Ανοίξτε ένα υπάρχον αρχείο σενάριο εντολών ή να δημιουργήσετε ένα νέο ώστε να επεξεργαστείτε.
Μπορείτε να έχετε δύο σενάρια που θα εκτελούνται πριν και μετά την μεταβολή των ρυθμίσεων.
Εάν χρειάζεστε μια χρονική καθυστέρηση μεταξύ της εκτέλεσης του σεναρίου και την αλλαγή των ρυθμίσεων δικτύου, μπορείτε να την ορίσετε στο κάτω μέρος.

Εκτέλεση προγράμματος

Το NetSetMan προσφέρει έναν έυκολο τρόπο για να εκτελέσετε εξωτερικά προγράμματα με την ενεργοποίηση του κάθε προφίλ.
Τα Προγράμματα μπορούν να εκκινηθούν είτε κάτω από χρήστη με περιορισμένα δικαιώματα (προκαθορισμένα με UAC ενεργό) είτε επαυξημένα από χρήστη με δικαιώματα διαχειριστή (χωρίς την παρέμβαση του UAC). Παράμετροι μπορούν να προστεθούν εφόσον απαιτείται.
Τα Προγράμματα μπορούν να εκκινηθούν είτε κανονικά, είτε μεγιστοποιημένα είτε τέλος ελαχιστοποιημένα.

Διαδικασίες οι οποίες ήδη εκτελούνται, μπορούν να τερματιστούν είτε βάση του ονόματος του αρχείου τους (άσχετα με το πού βρίσκονται) είτε βάση της ακριβής διαδρομής του αρχείου τους.

Κάθε ενέργεια μπορεί να έχει μια ανεξάρτητη/ξεχωριστή χρονική καθυστέρηση είτε πρίν (αρνητική τιμή) είτε μετά (θετική τιμή) την εκτέλεση της.

SMTP / Email

Σε ορισμένες περιπτώσεις, η διεύθυνση IP του διακομιστή SMTP για την αποστολή μηνυμάτων ηλεκτρονικού ταχυδρομείου ποικίλλει ανάλογα με το δίκτυό σας. Όντας στο ίδιο δίκτυο μπορεί να χρειαστείτε μια τοπική διεύθυνση IP, όπως πχ. 10.0.0.1, ενώ βρισκόμενοι εκτός του δικτύου μπορεί να χρειαστεί να έχετε πρόσβαση μέσω μιας παγκόσμιας διεύθυνσης IP.

Αντί να προσπαθείτε να είστε συμβατοί με πολλά διαφορετικά προγράμματα email και με διαφορετικά προφίλ χρηστών, το NetSetMan λύνει αυτό το πρόβλημα με μια απλή λύση. Ρυθμίστε το "nsm.smtp" ως διεύθυνση SMTP στο πρόγραμμα-πελάτη ηλεκτρονικού ταχυδρομείου σας και να ρυθμίσετε τις διευθύνσεις IP στο ανάλογο προφίλ. Όταν ενεργοποιείτε το προφίλ και οι δύο τιμές θα χρησιμοποιηθούν με τη χρήση του αρχείου hosts ώστε να μην χρειάζεται να κάνετε οποιεσδήποτε πρόσθετες αλλαγές στο πρόγραμμα-πελάτη ηλεκτρονικού ταχυδρομείου σας πια.

Ρυθμίσεις Συστήματος

Επιπρόσθετα με τις συνήθεις ρυθμίσεις δικτύου μερικές φορές μπορεί να θέλετε να αλλάξετε μερικές από τις ρυθμίσεις του συστήματος, ανάλογα με την τοποθεσία ή την κατάσταση σας. Για παράδειγμα, όσο βρίσκεστε στο πανεπιστήμιο μπορεί να θέλετε να απενεργοποιήσετε την ένταση του ήχου, να αλλάξετε την ταπετσαρία και να ενεργοποιήσετε το μοτίβο ελέγχου παροχής ενέργειας.

Μπορείτε επίσης να χρησιμοποιήσετε το παράθυρο Ρυθμίσεις Συστήματος για να αλλάξετε γρήγορα κάποια ρύθμιση που διαφορετικά θα σας έπαιρνε περισσότερο χρόνο προσπαθώντας να εντοπίσετε την εν λόγω ρύθμιση στα Windows. Απλά κάντε κλικ στο κουμπί κουμπί για να ενεργοποιήσετε άμεσα τη συγκεκριμένη ρύθμιση.

Η λίστα των υποστηριζόμενων ρυθμίσεων περιλαμβάνει: λειτουργία οθόνης, ταπετσαρία, το χρώμα της επιφάνειας εργασίας, screensaver, σύστημα σχεδιασμού, ηχεία και ένταση του μικροφώνου, σύστημα ήχου, το τείχος προστασίας (συμπεριλαμβανομένου εξαιρέσεις και τις κοινοποιήσεις.), Μοτίβο παροχής ενέργειας, ζώνη ώρας και τη διεύθυνση MAC.

WiFi

Ένα NetSetMan προφίλ μπορεί να χρησιμοποιηθεί για να συνδεθείτε σε ένα υπάρχον προφίλ WiFi. Για να το κάνετε αυτό, επιλέξτε έναν προσαρμογέα WiFi στη αναδυόμενη λίστα NICs και στη συνέχεια, επιλέξτε ένα προφίλ Wi-Fi από τη παραπάνω λίστα. Αν το μόνο που θέλετε είναι να αποσυνδέσετε τη σύνδεση WiFi σας, επιλέξτε την κενή εγγραφή.

Εκτός από αυτόν τον βασικό διακόπτη εναλλαγής WiFi, το NetSetMan προσφέρει μια εκτεταμένη διαχείριση WiFi. Κάντε κλικ στο εικονίδιο του WiFi ή χρησιμοποιήστε το κυρίως μενού (Εργαλεία> NSM Διαχείριση Ασύρματου Δικτύου) για να αποκτήσετε πρόσβαση. Θα βρείτε μια λίστα με όλα τα διαθέσιμα δίκτυα WiFi. Ένα κίτρινο φόντο υποδεικνύει ότι υπάρχει ήδη ένα προφίλ με το ίδιο SSID. Ένα δίκτυο με πράσινο φόντο είναι συνδεδεμένο. Μπορείτε να συνδεθείτε σε ένα δίκτυο με διπλό κλικ ή χρησιμοποιώντας το κουμπί "Σύνδεση". Αν δεν υπάρχει προφίλ για αυτήν τη σύνδεση, μπορείτε να επιλέξετε να το δημιουργήσετε πρώτα. Οι ρυθμίσεις (πιστοποίηση, κρυπτογράφηση, ...) θα είναι προ-ρυθμισμένες αυτόματα ανάλογα με το αναγνωρισμένο δίκτυο. Ωστόσο, ίσως χρειαστεί να τις προσαρμόσετε στην περίπτωση που η αναγνώριση δεν λειτούργησε σωστά για κάποιο λόγο.

Το NetSetMan λειτουργεί με τα εγγενή χαρακτηριστικά των Windows WiFi. Αυτό σημαίνει ότι έχετε πάντα πρόσβαση στο ίδιο προφίλ που ήδη χρησιμοποιείται στα Windows ή κάποιο άλλο λογισμικό WiFi. Μπορείτε να δημιουργήσετε, να τροποποιήσετε ή να διαγράψετε τα προφίλ WiFi.

Σύνδεση με MAC διεύθυνση

Τα Windows συνήθως συνδέονται σε ασύρματα δίκτυα βάση του SSID. Εάν πολλαπλά σημεία πρόσβασης μοιράζονται το ίδιο SSID, τα Windows τα ομαδοποιούν και σας συνδέουν σε εκείνο με το καλύτερο σήμα. Ωστόσω, μερικές φορές μπορεί να θέλετε να συνδεθείτε σε ένα συγκεκριμένο σημείο πρόσβαση. Με το NetSetMan μπορείτε να το κάνετε!

Το NSM WiFi Management εμφανίζει όλα τα διαθέσιμα δίκτυα καθώς και τα σημεία πρόσβασης χωρίς να τα ομαδοποιεί, ώστε να μπορείτε να κάνετε την σύνδεση που επιθυμείτε. Υπενθυμίσεις που εμφανίζονται με το πέρασμα του ποντικιού σας εμφανίζουν επιπρόσθετες πληροφορίες όπως την MAC διεύθυνση για κάθε δίκτυο. Το αναδυόμενο με δεξί κλικ μενού σα επιτρέπει να αντιγράψετε με ευκολία την MAC διεύθυνση ή το SSID.

Εάν επιθυμείτε να συνδεθείτε σε ένα συγκεκριμένο ασύρματο δίκτυο όταν ενεργοποιείτε ένα NetSetMan προφίλ, μπορείτε να το κάνετε προσθέντοντας την MAC διεύθυνση του στην λίστα των WiFi. Επιλέξτε “Προσθήκη…” και εισάγετε την MAC διεύθυνση της επιλογής σας. Κατά την ενεργοποίηση του προφίλ σας το NetSetMan θα συνδεθεί στο αντίστοιχο σημείο πρόσβασης. Σημειώστε, ότι εξακολουθεί να υπάρχει η ανάγκη ενός κατάλληλου προφίλ WiFi ρυθμισμένο στο σύστημα σας.

Απαιτήσεις

Το NetSetMan χρησιμοποιεί το γηγενές WiFi API των Windows. Αυτό του προσδίδει το μεγάλο πλεονέκτημα ότι δεν απαιτείτα εγκατάσταση νέων προγραμμάτων οδήγησης (τα οποία μπορεί να οδηγήσουν σε πολλές ασυμβατότητες). Από την άλλη πλευρά τα WiFi χαρακτηριστικά του NetSetMan λειτουργούν μόνον σε Windows συστήματα που υποστηρίζουν WiFi. Ευτυχώς, οι περισσότερες εκδόσεις που χρησιμοποιούνται στις μέρες μας υποστηρίζουν κάτι τέτοιο. Εάν χρησιμοποιείτε Vista ή νεότερη έκδοση, δεν έχετε κανένα πρόβλημα. Σε XP πρέπει να έχετε SP2 ή SP3 εγκατεστημένο.

Βεβαιωθείτε ότι η υπηρεσία WLAN AutoConfig (Wlansvc) (Vista ή μεταγενέστερο) ή Wireless Zero Configuration (WZCSVC) (XP) είναι σε λειτουργία.

Εάν ο προσαρμογέας ασύρματου δικτύου σας έρχεται με λογισμικό κατασκευαστή για να διαχειριστείτε τις ρυθμίσεις WiFi, τότε αυτό θα εμποδίσει την πρόσβαση των Windows, στις περισσότερες περιπτώσεις. Θα πρέπει να βρείτε την επιλογή ώστε να κάνετε χρήση της διαχείρισης των Windows σε αυτό το λογισμικό (εάν υπάρχει).
Επειδή υπάρχουν πολλοί διαφορετικοί κατασκευαστές με το δικό τους λογισμικό, δεν υπάρχει καθολικός τρόπος/οδηγία για να επιτευχθεί αυτό. Παράδειγμα: Εάν χρησιμοποιείτε μια AVM FRITZ!WLAN USB συσκευή, κάντε κλικ στο εικονίδιο του λογισμικού της στον χώρο ειδοποιήσεων και επιλέξτε "Windows WLAN".

Προηγμένο
Αυτόματη Εναλλαγή

Είναι αρκετά εύκολο να ενεργοποιήσετε ένα προφίλ με ένα κλικ από το μενού στον χώρο ειδοποιήσεων . Αλλά σε ορισμένες περιπτώσεις μπορεί να είναι ακόμα πιο εύκολο να ενεργοποιήσετε ένα προφίλ εντελώς αυτόματα ανάλογα με τις συνθήκες. Το ενσωματωμένο χαρακτηριστικό Αυτόματης Εναλλαγής κάνει ακριβώς αυτό.

Μετά την ενεργοποίηση Αυτόματης Εναλλαγής οι παρακάτω συνθήκες μπορούν να ρυθμιστούν για όλα τα προφίλ ξεχωριστά:

Γειτονικά Δίκτυα WiFi
  • Ασύρματο δίκτυο (SSID ή MAC) εντός / εκτός βεληνεκούς
  • Η ισχύς σήματος του ασύρματου δικτύου είναι χαμηλότερα / υψηλότερα από ό, τι μια ορισμένη τιμή
Τοποθεσίες Δίκτυου
  • Windows NLM: Αυτόματη δημιουργία προφίλ δικτύου από τα Windows (Vista ή μεταγενέστερα)
  • Πύλη MAC: αποτύπωμα του αποδίδεται αυτόματα σε πύλη DHCP
  • Ping: Διαθεσιμότητα ενός συγκεκριμένου εξυπηρετητή ή συστήματος στο δίκτυο
Κατάσταση Προσαρμογέα Δικτύου
  • Κατάσταση σύνδεσης του προσαρμογέα τοπικού δικτύου (σύνδεση / αποσύνδεση)
Χρονοδιάγραμμα
  • Ορισμένες ημέρες της εβδομάδας και το χρονικό πλαίσιο (π.χ. Σα + Κυ, 8:00 - 18:00)

Αν και δεν υπάρχει τεχνικός περιορισμός για τον αριθμό των συνθηκών ανά προφίλ προσπαθήστε να το διατηρείτε όσο το δυνατόν απλούστερο. Στις περισσότερες περιπτώσεις είναι απολύτως επαρκές να περιλαμβάνετε ένα ή δύο συνθήκες για να επιτευχθεί το επιθυμητό αποτέλεσμα.

Μπορείτε να καθορίσετε ότι είτε το σύνολο είτε τουλάχιστον μία από τις συνθήκες που πρέπει να πληρούνται θα απαιτούνται για την ενεργοποιήσετε του προφίλ.

Σημείωση: Τα προφίλ με χαμηλότερους κωδικούς έχουν υψηλότερη προτεραιότητα σε περίπτωση ταυτόχρονης εφαρμογής πολλαπλών προφιλ.

Από προεπιλογή, η διαδικασία ενεργοποίησης ξεκινά με ένα μήνυμα στην κάτω δεξιά γωνία με μια προσαρμόσιμη καθυστέρηση αντίστροφης μέτρησης, η οποία σας επιτρέπει προαιρετικά να διακόψετε τη διαδικασία ενεργοποίησης. Εναλλακτικά, είτε μια ειδοποίηση των Windows μπορεί να χρησιμοποιηθεί είτε το προφίλ να ενεργοποιηθεί χωρίς κανένα μήνυμα στο παρασκήνιο. Στην τελευταία περίπτωση, το εικονίδιο της γραμμής θα κινείται κατά τη διαδικασία ενεργοποίησης και θα αλλάξει το χρώμα του προσωρινά σύμφωνα με το αποτέλεσμα ενεργοποίησης για να σας ενημερώσει οπτικά σχετικά με την αυτοματοποιημένη διαδικασία.

Για να εξοικονομήσετε πόρους συστήματος το NetSetMan δεν συγκρίνει τις τρέχουσες ρυθμίσεις σας με τις ρυθμίσεις του προφίλ που πρόκειται να ενεργοποιηθεί. Αυτό σημαίνει ότι αν αλλάξετε οποιαδήποτε από τις ρυθμίσεις σας με το χέρι, το NetSetMan δεν θα παρατηρήσει την αλλαγή.

Η Αυτόματη Εναλλαγή έχει κάποιους περιορισμούς στα XP: Τα Windows NLM δεν είναι διαθέσιμα, Αλλαγές στα γειτονικά δίκτυα WiFi ενδέχεται να αναγνωριστούν με σημαντική καθυστέρηση.

Σημειώσεις

Μπορείτε να δημιουργήσετε σύντομες σημειώσεις για κάθε προφίλ αν πρέπει να θυμηθείτε ή να αναφέρετε τον σκοπό του προφίλ. Εάν ένα σημείωμα δημιουργήθηκε για ένα προφίλ, τότε το εικονίδιο του θα εμφανίζεται έγχρωμο στην μπάρα του μενού. Η σημείωση είναι τότε ορατή σε πλήρη λειτουργία ως έναν υπαινιγμό της εικόνα και στο Συμπαγή Λειτουργία στην κορυφή των ρυθμίσεων του προφίλ.

NSM Υπηρεσία / Χρήστης περιορισμένης πρόσβασης

Το NetSetMan απαιτεί προνόμια διαχειριστή για να είστε σε θέση να αλλάξει τις ρυθμίσεις του δικτύου και του συστήματος για εσάς. Κανονικά η εκτέλεση μιας τέτοιας εφαρμογή δεν είναι δυνατή για έναν χρήστη με περιορισμένη πρόσβαση. Νεότερες εκδόσεις των Windows, περιορίζουν ακόμη και τους λογαριασμούς διαχειριστή με το UAC (User Account Control) εμφανίζοντας ένα μήνυμα σε κάθε εκκίνηση του προγράμματος.

Για να λύσει το ζήτημα αυτό το NetSetMan περιλαμβάνει ένα ειδικό συστατικό που ονομάζεται NSM Υπηρεσία η παρέχει τα απαιτούμενα δικαιώματα. Η υπηρεσία εγκαθίσταται αυτόματα, αλλά μπορεί επίσης να εγκατασταθεί / απεγκατασταθεί χειροκίνητα στις προτιμήσεις (Προνόμια Συστήματος> Ενεργοποίηση / Απενεργοποίηση).

Η εγκατάσταση της υπηρεσίας είναι ορατή στο παράθυρο Υπηρεσιών των Windows (services.msc) και μπορεί να ελέγχθεί και από εκεί. Επιπρόσθετα, μπορεί να εγκατασταθεί / απεγκατασταθεί ή να ξεκινήσει / σταματήσει μέσω της χρήσης γραμμών εντολών:
nsmservice.exe /install /silent
nsmservice.exe /install /silent
net start nsmService
net stop nsmService

Εάν πάρετε το σφάλμα "Άκυρη υπογραφή", βεβαιωθείτε ότι το exe του προγράμματος δεν έχει τροποποιηθεί και η πολιτική ασφαλείας ομάδων των Windows "Turn off Automatic Root Certificates Update" είναι απενεργοποιημένη.

Εάν εκτελείτε το NetSetMan με έναν λογαριασμό περιορισμένης πρόσβασης, όπου η Υπηρεσία NSM δεν είναι διαθέσιμη, θα λάβετε ένα μήνυμα που εξηγεί το πρόβλημα. Ως εναλλακτική λύση μπορείτε να χρησιμοποιήσετε πιστοποιήση διαχειριστή από έναν τοπικό λογαριασμό που το NetSetMan θα αποθηκεύσει κρυπτογραφημένα στο αρχείο ρυθμίσεων και να εκκινήσετε το πρόγραμμα με τα απαιτούμενα δικαιώματα. Παρατηρήστε ότι στην περίπτωση αυτή η εφαρμογή θα εκτελείται στο πλαίσιο του χρήστη του οποίου τα διαπιστευτήρια χρησιμοποιήθηκαν.

Προ-σύνδεσης Εναλλαγή Προφίλ

Ένα πολύ μοναδικό και ισχυρό χαρακτηριστικό γνώρισμα είναι η δυνατότητα να ενεργοποιήσετε τα προφίλ σας στην οθόνη σύνδεσης. Αυτό επιτρέπει την προετοιμασία των σωστών ρυθμίσεων του δικτύου που χρειάζεστε για το λογαριασμό σας ενεργοποιώντας ένα NetSetMan προφίλ, ακόμη και πριν την σύνδεση.

Για να χρησιμοποιήσετε αυτή τη δυνατότητα η NSM Υπηρεσία πρέπει να εγκατασταθεί και η επιλογή «Προφίλ μετάβαση από την οθόνη σύνδεσης των Windows" (Preferences> Προνόμια Συστήματος) πρέπει να είναι ενεργοποιημένη.

Σημειώστε ότι αυτό το χαρακτηριστικό σας επιτρέπει μόνο να αλλάξετε τις ρυθμίσεις που είναι κοινές μεταξύ όλων των χρηστών. Συγκεκριμένες ρυθμίσεις χρήστη δεν μπορούν να αλλάξουν έξω από το πλαίσιο του χρήστη.

NSM Διαχείριση

Εάν ρυθμίσετε το NetSetMan ώστε να χρησιμοποιηθεί από κάποιον άλλο, ίσως να θέλετε να καθορίσετε τις ενέργειες που το άλλο άτομο έχει τη δυνατότητα να εκτελέσει. Με την NSM Διαχείριση (Προτιμήσεις> Διαχείριση) μπορείτε να κάνετε ακριβώς αυτό.

Όταν ενεργοποιείτε αυτή τη λειτουργία θα πρέπει να εισάγετε έναν κωδικό πρόσβασης με 6 ή περισσότερους χαρακτήρες. Μόνο με αυτόν τον κωδικό πρόσβασης θα είστε σε θέση να αλλάξετε τις ρυθμίσεις του NSM Διαχείριση ή να εκτελέσετε τις ενέργειες που επιλέξατε να ασφαλιστούν (εκτέλεση NetSetMan, ενεργοποίηση προφίλ, αλλαγή ρυθμίσεων προφίλ, αλλαγή ρυθμίσεων του προγράμματος).

Μπορείτε να κλειδώσετε ακόμη ορισμένα προφίλ έτσι ώστε ο χρήστης να μην μπορεί να τα τροποποιήσει, αλλά να εξακολουθεί να είναι σε θέση να τα ενεργοποιήσει. Ή μπορείτε να περιορίσετε γενικά την πρόσβαση σε ορισμένες ρυθμίσεις όπως "Ομάδα Εργασίας & Τομέας", ενώ να επιτρέψετε στο χρήστη να μεταβάλλει όλες τις άλλες ρυθμίσεις.

Ένα άθροισμα ελέγχου του συνθηματικού σας θα εγγραφεί στο μητρώο. Βεβαιωθείτε ότι ένας κανονικός χρήστης δεν έχει δικαιώματα εγγραφής στο HKLM \ Software \ NetSetMan (για x64: HKLM \ Software \ Wow6432Node \ NetSetMan). Αν ξεχάσετε τον κωδικό σας, θα πρέπει να διαγράψετε χειροκίνητα το κλειδί. Αν το αρχείο ρυθμίσεων σας είναι κρυπτογραφημένο τότε κανείς χωρίς τον κωδικό, δεν θα μπορεί ούτε να το δει αλλά ούτε και να το επεξεργαστεί.

Ενεργοποίηση μέσω Γραμμής Εντολών

Σε ορισμένες ειδικές περιπτώσεις, μπορεί να θέλετε να αλλάξετε το προφίλ σας, χωρίς τη χρήση του GUI του NetSetMan: αλλαγή προφίλ με συντομεύσεις, ενεργοποιήση ενός προφίλ κατά την εκκίνηση, συμπερίληψη σε ένα πολύπλοκο σενάριο εντολών, κ.λπ.

Οι παρακάτω παράμετροι είναι διαθέσιμες:

-a # Ενεργοποιεί το προφίλ με αριθμό #
-as # Ενεργοποιεί το προφίλ σε αθόρυβη λειτουργία (το παράθυρο ενεργοποίησης θα κλείσει αυτόματα αν δεν εμφανιστούν σφάλματα)
-h / -hide Εκτέλεση του NetSetMan ελαχιστοποιημένο
-f / -full Εκτέλεση του NetSetMan σε Πλήρη Λειτουργία (προεπιλογή)
-c / -compact Εκτέλεση του NetSetMan στην Συμπαγή Λειτουργία
-m / -multi Εκτελεί ένα δεύτερο αντίτυπο του NetSetMan. Να είστε προσεκτικοί: Όλες οι ενεργές εκτελέσεις του NetSetman γράφουν στο ίδιο αρχείο ρυθμίσεων!

Μια συντόμευση για την ενεργοποίηση του προφίλ # 0 σε σιωπηλή λειτουργία θα μπορούσε να μοιάζει με αυτό:
"C:\Program Files (x86)\NetSetMan\netsetman.exe" -as 0

Ομάδες Προφίλ

Με το NetSetMan Pro είστε σε θέση να δημιουργήσετε απεριόριστο αριθμό ομάδων προφίλ. Κάθε τέτοια ομάδα περιλαμβάνει 6 προφίλ.

Για να μετακινηθείτε μεταξύ όλων των διαθέσιμων ομάδων προφίλ επιλέξετε οποιαδήποτε ομάδα από το πτυσσόμενο πεδίο στη γραμμή μενού.

Εάν έχετε περισσότερες από μία ομάδες, τα προφίλ σας θα εμφανίζονται ομαδοποιημένα στο μενού του χώρου ειδοποιήσεων.

Περιηγητής / Μεσολαβητής

Με το NetSetMan Pro μπορείτε να διαχειριστείτε τις ρυθμίσεις του διακομιστή μεσολάβησης των Windows / Internet Explorer, Firefox και Opera (<= v12). Chrome και νεότερες εκδόσεις του Opera δεν υποστηρίζουν τις δικές σας ρυθμίσεις μεσολαβητή, αλλά αντ 'αυτού χρησιμοποιειούν αυτόματα τις ρυθμίσεις του συστήματος.

Όλες οι ρυθμίσεις προορίζονται να μιμούνται τις αρχικές διεπαφές για να επιτραπεί η ίδια διάταξη που υπάρχει στα προγράμματα περιήγησης άμεσα. Επιπλέον, μπορείτε να αλλάξετε την αρχική σελίδα και στην περίπτωση που χρησιμοποιείτε προφίλ στο Firefox ή Opera θα είστε επίσης σε θέση να επιλέξετε το όνομα του προφίλ.

Παρακαλούμε σημειώστε ότι ο Firefox πρέπει να κλείσει πριν αλλάξετε τις ρυθμίσεις του.
Εάν έχετε πολλά προφίλ Firefox και ο φάκελος των προφιλ αλλάξει (π.χ., λόγω του νέου συστήματος), θα πρέπει να επιλέξετε ξανά το προφίλ στη λίστα ώστε να ενημερωθεί το αρχείο ρυθμίσεων σας.

Τομέας

Για να συμμετάσχετε σε έναν τομέα εισάγετε το όνομά του, το όνομα χρήστη και τον κωδικό πρόσβασής σας. Το όνομα χρήστη και ο κωδικός πρόσβασής θα αποθηκευτούν κρυπτογραφημένα στο αρχείο ρυθμίσεων.

Συνήθως είναι αρκετό να δώσετε άμεσα το όνομα χρήστη. Ωστόσο, εάν λαβετε ένα σφάλμα, μπορείτε να δοκιμάσετε δίνοντας το όνομα χρήστη με ένα από τα παρακάτω πρότυπα:
a) NETBIOS: DOMAIN\user
β) UPN: user@domain.local

Σε NT τομείς ένας χρήστης μπορεί να είναι μέλος σε ένα τομέα μόνο. Αυτό σημαίνει ότι με την ένταξη του σε ένα άλλο τομέα ή μια ομάδα εργασίας η συμμετοχή του στον προηγούμενο τομέα έχει χαθεί μέχρι το σύστημά σας να επανενωθεί με τον τομέα.

Το NetSetMan μπορεί να συνδέσει σε έναν τομέα, μόνο εάν ο λογαριασμός χρήστη τομέα σας έχει την άδεια να το πράξει.
Αυτός το προνόμιο του λογαριασμού χρήστη ονομάζεται "Προσθήκη σταθμού εργασίας σε τομέα" και πρέπει να ενεργοποιηθεί για τον λογαριασμό σας. Αν δεν έχετε αυτό το προνόμιο, ζητήστε από τον διαχειριστή του δικτύου σας να σας το χορηγήσει.
