Descrizione

NetSetMan è un gestore di impostazioni di rete che può passare facilmente tra 6 diversi profili visivamente strutturati, tra cui:

  • Indirizzi IP (IPv4 e IPv6!)
  • Gateway (incl. Metric)
  • Server DNS
  • Server WINS
  • Gestione WiFi
  • Nome computer
  • Gruppo di lavoro
  • Stampante predefinita
  • Unità di rete
  • NIC Stato
  • Connessioni (RAS / DUN / PPP / VPN)
  • Suffisso DNS
  • Tabella Percorso
  • Server SMTP
  • Hosts File Entries
  • Impostazioni di sistema
  • MAC Address
  • Script (BAT, VS, JS, ...)
Inoltre, solo nella versione Pro:
  • Può essere usato durante il lavoro
  • Quantità illimitata di profili
  • Network-Dominio
  • Completa Impostazioni Proxy &
    Bowser Pagina
    (Internet Explorer, Firefox, Opera)

NetSetMan offre ciò che vi siete persi in Windows fino ad ora:
Una potente interfaccia, facile da usare per gestire tutte le impostazioni di rete a colpo d'occhio.
Fate una prova e non vorrete mai essere senza di nuovo.

Tutte le impostazioni e le modifiche vengono salvate automaticamente in un file separato (settings.ini) così una copia di backup, il trasferimento o l'aggiornamento a un'altra versione è assolutamente un problema.

Attivare un profilo memorizzato con un solo clic o addirittura completamente automatico!

Informazioni di Base
  • Profili
    Ogni profilo rappresenta un insieme di impostazioni necessarie per una determinata località o scenario. Utilizzando sottoprofili più schede di rete può essere configurato con un profilo. Rinominare un profilo facendo doppio clic sulla relativa scheda didascalia.
  • Configurazione manuale
    È necessario configurare i profili richiesti manualmente voi stessi. Non è possibile indovinare le impostazioni corrette per una località sconosciuta. Lo scopo principale dell'applicazione è quello di automatizzare le modifiche delle impostazioni regolari che ti sentiresti di solito fare manualmente.
  • Selezione delle impostazioni
    Leggere tutte le impostazioni correnti in un profilo in grado di dare un punto di partenza, ma è ancora necessario decidere quali impostazioni da applicare quando si attiva tale profilo. Si raccomanda di limitare le impostazioni a quelle che potrebbe essere stato effettivamente cambiato nel frattempo. Ad esempio, non è necessario applicare un nome computer se questa impostazione non viene mai modificata.
  • Configurazione di impostazioni
    Per configurare verificare l'impostazione relativa casella di controllo prima, allora o immettere la configurazione desiderata direttamente nell'interfaccia principale o aprire la finestra di configurazione per quella particolare impostazione.
  • Attivazione Profilo
    Per impostazione predefinita, nessun profilo viene attivato automaticamente, vale a dire nessuna impostazione sul vostro sistema è cambiato, senza la vostra intenzione. Detto questo, è possibile utilizzare la AutoSwitch funzione per attivare determinati profili automaticamente in base alle condizioni di liberamente personalizzabili.
  • Vassoio accesso icona
    Si consiglia di tenere NetSetMan minimizzato nel vassoio e spostare l'icona del vassoio nella zona sempre visibile. In questo modo è possibile attivare rapidamente un profilo facendo clic sull'icona e selezionando un profilo. Inoltre avrete una panoramica utile sopra le vostre IP di connessione attuali posizionando il mouse sopra l'icona con il cursore del mouse.
Profili

Le schede rappresentano i profili disponibili. È possibile salvare 6 diversi gruppi di impostazioni in quelle schede.

Se vuoi modificare le impostazioni di due schede di rete differenti quando si attiva un singolo profilo, è possibile utilizzare la funzione SUB. Fare clic su "+" - pulsante accanto al menu a discesa e una nuova barra di menu apparirà contenente due sottoprofili (quello attuale e uno nuovo vuoto). La voce sarà automaticamente il nome della scheda di rete selezionata in quel sottoprofilo e non può essere modificato. È possibile creare il numero di sottoprofili che vuoi. Tutti loro saranno attivati ​​uno dopo l'altro quando si attiva il profilo. È possibile modificare l'ordine con il drag-and-drop.

Tutte le modifiche vengono salvate automaticamente in modo niente si perde accidentalmente.

È possibile rinominare un profilo facendo clic destro su di esso e scegliere "Rinomina" o semplicemente facendo doppio clic sulla relativa scheda didascalia.

Per gestire tutti i tuoi profili più facilmente si può utilizzare il Gestione profilo che troverete nella barra dei menu. Esso permette di modifica dei profili, sottoprofilo e gruppi (solo in NetSetMan Pro) con il mouse o alcune scorciatoie da tastiera comune (tasti freccia per la navigazione, F2 per la ridenominazione, ecc). Quando si usa il drag-and-drop per riorganizzare gli elementi noterete alcune restrizioni inevitabili seconda della voce, a causa di determinate circostanze tecniche: i profili non possono essere cancellati, MultiProfiles non possono essere rinominati, profili possono essere scambiati, ma non spostati. Le caselle di controllo rappresentano se il profilo base sarà visibile in vassoio e in modalità compatta.

Schede di rete (NIC)

Tutte le schede di rete disponibili sono visualizzate con il loro nome e il loro tipo. Selezionare la scheda NIC vuoi gestire nel (sotto) del profilo attuale.

Se una determinata scheda di rete non è disponibile nell'elenco (ad esempio, perché è disattivato o virtuale) è possibile gestire l'elenco facendo clic sul pulsante Aggiorna e sia rapidamente aggiornare l'elenco o scegliere Gestisci schede di rete. Vi troverete un elenco completo di tutte le schede di rete disponibili. Controllare tutte le schede di rete che si desidera avere nella vostra lista.

Nota: Tutte le schede di rete che sono inclusi in tale elenco apparirà anche nel Info vassoio. NIC indesiderati deselezionare per rimuoverli dalla lista e l'Info vassoio.

Se si copia il file di impostazioni (settings.ini) ad un altro sistema, NetSetMan proverà a riassegnare automaticamente le schede di rete nei profili. Tuttavia, questo funziona solo se il nuovo sistema ha una scheda di rete con esattamente lo stesso nome (nel primo screenshot è "LAN").

Impostazioni Operazioni

Ottenere Impostazioni correnti

Per caricare una impostazione corrente nel profilo basta cliccare sul simbolo accanto al campo di immissione.
Per ottenere tutte le impostazioni correnti fare clic destro sul profilo e scegliere "Visualizza impostazioni correnti".

Impostazioni Clearing

Per cancellare la configurazione di un certo ambiente destro del mouse e scegliere "Cancella".

Per cancellare i valori di un campo IP sufficiente fare doppio clic sul suo didascalia corrispondente.

Per cancellare tutte le impostazioni della corrente (sub) profilo destro del mouse sulla scheda profilo e scegliere "Impostazioni Cancella".

Copia & Incolla Impostazioni

Per trasferire le impostazioni tra i diversi profili è possibile copiare e incollare un profilo intero facendo clic con il profilo originale, scegliere "Impostazioni copia", poi clic destro sul nuovo profilo e scegliendo "Incolla Impostazioni".

Per copiare e incollare solo una specifica impostazione semplicemente fare clic destro e scegliere "Copia" o "Incolla".

Attivazione Profilo

Per applicare tutte le impostazioni del profilo corrente cliccare su "Attiva", o se il programma è in esecuzione minimizzato selezionare il profilo dal menu della barra .
Si noti che saranno attivate solo le impostazioni con una casella di controllo selezionata.

La finestra di dialogo di attivazione contiene le informazioni sullo stato di avanzamento di attivazione corrente.
Ci si può vedere quali impostazioni stanno per essere applicate e se il processo va a buon fine o meno. È possibile annullare il processo di attivazione, fare clic sul pulsante "Annulla" in qualsiasi momento.

Tutte le modifiche vengono applicate immediatamente. Tuttavia, in alcuni casi, alcune impostazioni prendere un po 'di tempo per prendere pieno effetto o richiedere un riavvio (in particolare "PC Nome" e "Gruppo di lavoro / Dominio").

La sezione Dettagli fornisce una panoramica di tutte le impostazioni configurate per tale profilo. Per maggiori informazioni e messaggi di errore in tutta la lunghezza aprire la sezione Registro.

Per trovare soluzioni per gli errori ei problemi più comuni hanno un'occhiata al FAQ .

Modo & Modo Compatto completa

Oltre alla classica interfaccia "Full Mode", in cui è possibile modificare tutte le impostazioni di un profilo, un'interfaccia aggiuntiva "Modalità Compact" è disponibile. È possibile passare tra queste modalità in qualsiasi momento facendo clic sul pulsante funzione nel menu principale sulla destra.

Una volta che tutti i profili sono impostati la modalità Compact è l'interfaccia ideale. Qui si ha una panoramica compatta di tutte le impostazioni di un profilo senza doversi preoccupare delle impostazioni vengano modificati accidentalmente.

Nel caso in cui si utilizza l' Amministrazione NSM (con "Modificare le impostazioni del profilo" sia deselezionata) la modalità Compact sarà usato automaticamente per impedire all'utente di essere in grado di modificare le impostazioni.

Menu Vassoi & Info Tray

Quando NetSetMan è in esecuzione troverete la sua icona nella barra di sistema (accanto all'orologio).
Non è possibile attivare rapidamente un profilo. Basta fare clic destro sull'icona per aprire il menu Vassoi e scegliere il profilo da attivare.

Se vuoi escludere alcuni profili di apparire nel vassoio, è possibile nascondere singolarmente. Per fare questo selezionare il profilo nella finestra principale, fare clic destro sulla scheda e fare clic su "Attivo in modalità vassoio e Compact". Il piccolo simbolo a fianco il testo indica se questo profilo sarà disponibile nel vassoio. Questa opzione avrà anche effetto sui profili indicati nella modalità compatta. Profilo visibilità può essere configurata anche in Gestione profilo .

Se si posiziona il cursore sull'icona NetSetMan nella barra apparirà Info vassoio. Esso contiene le informazioni IP corrente che viene utilizzato dal sistema. Dato che i dati vengono raccolti in tempo reale, si potrebbe verificare un breve ritardo. Per ridurre il ritardo che si dovrebbe considerare disattivare informazioni aggiuntive come IP pubblico e gli indirizzi MAC in Preferenze.

Caratteristiche
Panoramica

NetSetMan è stato progettato in un modo che permette di avere una visione diretta di tutte le impostazioni configurate all'interno di un profilo. Alcune delle caratteristiche più importanti come le impostazioni IP di base, WiFi e la stampante predefinita possono essere configurati direttamente sull'interfaccia principale. Altre impostazioni, più complesse possono essere attivati ​​solo qui e mostrerà una breve panoramica sulla loro configurazione attuale. Per modificare le loro attivano prima, quindi aprire la finestra di dialogo in base cliccando il [...] - pulsante.

Le sezioni seguenti vi daranno le istruzioni di base e consigli utili su come utilizzare tutte le funzionalità.

Adattatore di stato

Per attivare, disattivare o riavviare la scheda di rete selezionata utilizzare il tasto adattatore di stato. Il colore predefinito cambierà in un colore di stato connessi. Inoltre, è possibile visualizzare lo stato configurato nel tooltip.

Se si desidera attivare un adattatore e disabilitare uno o tutti gli altri adattatori, è necessario creare sottoprofili per ciascuna delle schede. È possibile eseguire questa operazione manualmente o utilizzando la voce di menu nella parte inferiore del menu Adapter Stato. Si creerà automaticamente un sottoprofilo per ogni scheda di rete che è incluso nella lista adattatore e impostare l'impostazione di conseguenza il suo status. Ricorda che è possibile personalizzare quali adattatori sono inclusi in tale elenco utilizzando la scheda di gestione della rete .

Impostazioni avanzate TCP / IP

È possibile specificare più IP, Gateway, DNS e WINS indirizzi per ogni scheda di rete, se avete bisogno di essere in diverse reti allo stesso tempo. Per farlo clicca sul pulsante "IP +" o scegliere "Profilo> Impostazioni avanzate TCP / IP" dal menu principale.

Qui avete anche la possibilità di gestire le impostazioni più avanzate come la tabella di routing, suffisso DNS, IPConfig e NetBIOS.

  • Attivare un gruppo verificando è casella di didascalia
  • Aggiungere un nuovo valore
  • Eliminare il valore selezionato
  • Cancellare l'intera lista
  • Ottenere tutti i valori di corrente per questa impostazione del sistema (considerando che cliccando sul simbolo nella finestra principale legge il primo valore solo)
  • Riorganizzare l'ordine con il drag-and-drop
Collegamenti

Utilizzare questa funzione per collegare o scollegare qualsiasi tipo di connessioni Windows come reti dial-up, banda larga mobile o VPN. Connessioni compatibili sono elencati in Windows insieme con i vostri normali schede di rete nella finestra di dialogo Connessioni di rete. Da un punto di vista tecnico tutte le connessioni che sono compatibili con "RasDial" sono supportati.

È possibile scegliere di disconnettere tutte le connessioni attualmente stabilite selezionando la casella di controllo, o per collegare / scollegare alcuni collegamenti singolarmente in ordine di lista.

Host

In casi molto specifici potrebbe essere necessario o utile per assegnare manualmente indirizzi IP ai nomi o domini. Si noti che NetSetMan solo modificare le voci propri ospiti lasciando le voci già esistenti intatta.

Unità di rete

Con NetSetMan è possibile mappare ed eliminare la mappatura di un numero illimitato di unità di rete.
Si prega di leggere le note di istruzioni nella parte inferiore della finestra cliccando su "[Show Aiuto]".

Quando viene attivato un profilo e la funzione Net unità è selezionata NetSetMan confronterà l'elenco di tutte le unità attualmente mappate con l'elenco delle unità da mappare. Tutte le unità mappate che sono in entrambe le liste (stessa lettera di unità e lo stesso percorso di posizione) rimarranno intatti. Tutte le altre unità di rete saranno mappata prima che quelli nuovi sono mappati.

Script

NetSetMan offre un modo comodo per gestire i file di script. Tuttavia, non ha proprio linguaggio di script. Questo significa che è possibile utilizzare solo i linguaggi di script il sistema operativo è in grado di eseguire. Linguaggi di script comuni sono VBS, JS e file batch da riga di comando anche semplici. Usa il motore di ricerca preferito per trovare ulteriori informazioni sugli script e cosa si può fare con esso.

Aprire un file di script esistente o crearne uno nuovo per essere in grado di modificare lo script.
È possibile avere due script che verranno eseguiti prima e dopo tutte le altre impostazioni vengono modificate.
Se avete bisogno di un ritardo tra l'esecuzione dello script e la modifica delle impostazioni di rete, è possibile specificare in fondo.

SMTP / Email

In alcuni casi l'indirizzo IP del server SMTP per l'invio di messaggi di posta elettronica varia a seconda della rete. Essere nella stessa rete potrebbe essere necessario un indirizzo IP locale come 10.0.0.1, pur essendo al di fuori di tale rete potrebbe essere necessario accedervi da un indirizzo IP globale.

Invece di cercare di essere compatibile con un sacco di diversi client di posta elettronica e si occupano di diversi profili utente NetSetMan risolve questo problema con una soluzione semplice. Impostare "nsm.smtp" come indirizzo SMTP nel tuo client di posta elettronica e configurare gli indirizzi IP nei profili secondo. Quando si attiva il profilo di entrambi i valori saranno collegati utilizzando il file hosts in modo che non c'è bisogno di fare ulteriori modifiche nel vostro client di posta elettronica più.

Impostazioni di sistema

In aggiunta alle impostazioni di rete soliti a volte si potrebbe voler modificare alcune impostazioni di sistema a seconda della vostra posizione o situazione. Ad esempio, mentre all'università si potrebbe desiderare di disattivare il volume, cambiare lo sfondo e attivare il piano di alimentazione equilibrata.

È inoltre possibile utilizzare la finestra Impostazioni di sistema per cambiare rapidamente alcune impostazioni che altrimenti richiedere più tempo per trovare la posizione di tale impostazione in Windows. Basta fare clic sul pulsante per attivare direttamente tale impostazione particolare.

Impostazioni supportati comprendono: modalità di visualizzazione, carta da parati, il colore del desktop, screensaver, schema di disegno, altoparlanti e volume del microfono, combinazione di suoni, firewall (inclusa eccezioni e notifiche.), Il risparmio di energia, il fuso orario e l'indirizzo MAC.

Wifi

Un profilo NetSetMan può essere utilizzato per connettersi a un profilo Wi-Fi esistente. Per fare questo, selezionare un adattatore WiFi nella lista NIC a discesa, quindi selezionare un profilo Wi-Fi dall'elenco a discesa WiFi. Se si desidera solo per disconnettere la connessione WiFi, scegliere la voce vuota.

Oltre a questo interruttore WiFi di base NetSetMan offre una gestione estensiva WiFi. Fare clic sull'icona WiFi o utilizzare il menu principale (Strumenti> Gestione WiFi NSM) per accedervi. Troverete un elenco di tutte le reti WiFi attualmente disponibili. Uno sfondo giallo indica che un profilo con lo stesso SSID esiste già. Una rete con uno sfondo verde è attualmente collegato. È possibile connettersi a una rete facendo doppio clic su di esso o con il "Connect" -Button. Se non c'è alcun profilo esistente per questa connessione è possibile scegliere di creare in primo luogo. Le impostazioni (autenticazione, crittografia, ...) saranno pre-configurati automaticamente a seconda della rete riconosciuta. Tuttavia, potrebbe essere necessario regolarli nel caso in cui il riconoscimento non ha funzionato correttamente per qualche motivo.

NetSetMan funziona con i profili di Windows WiFi nativo. Ciò significa che si ha sempre accesso agli stessi profili che hai già utilizzato in Windows o qualche altro software WiFi. È possibile creare, modificare o eliminare i profili Wi-Fi.

 

Requisiti

NetSetMan utilizza l'API WiFi nativo di Windows. Che porta il grande vantaggio che nessun nuovo driver devono essere installati (che potrebbe tradursi in diversi conflitti). Il lato negativo le caratteristiche WiFi di NetSetMan funzioneranno solo su sistemi Windows che già supportano WiFi. Fortunatamente la maggior parte delle versioni utilizzate al giorno d'oggi lo fanno. Se si utilizza Vista o poi si sta bene ad andare. In XP è necessario disporre di SP2 o SP3 installato.

Assicurarsi che il servizio Configurazione automatica WLAN (Wlansvc) (Vista e versioni successive) o Configurazione Wireless Zero (WZCSVC) (XP) è in esecuzione.

Se la scheda WiFi viene fornito con un software costruttore personalizzato per gestire le impostazioni WiFi, allora sarà bloccare l'accesso di Windows, nella maggior parte dei casi. Avrete bisogno di trovare la possibilità di passare alla gestione di Windows in quanto il software (se disponibile).
Perché ci sono molti produttori diversi con il proprio software, non c'è modo universale per raggiungere questo obiettivo. Ecco un esempio: Se si utilizza un AVM FRITZ!Dispositivo USB WLAN, fare clic sull'icona del vassoio del suo software e scegliere "Windows WLAN".

Avanzate
AutoSwitch

E 'abbastanza facile da attivare un profilo con un solo clic dal menu Vassoi . Ma in alcuni casi potrebbe essere ancora più facile attivare i profili in modo completamente automatico a seconda delle condizioni personalizzate. La funzione integrata AutoSwitch fa esattamente questo.

Dopo aver attivato AutoSwitch possono essere configurati per tutti i profili individualmente le seguenti condizioni:

Reti wireless circostanti
  • Rete wireless (SSID o MAC) in / fuori range
  • La potenza del segnale di una rete wireless inferiori / superiori a un certo valore
Percorsi di rete
  • Finestre NLM: generato automaticamente profili di rete da Windows (Vista e versioni successive)
  • Gateway MAC: impronte digitali del gateway DHCP assegnato automaticamente
  • Ping: Disponibilità di un determinato server o un sistema in rete
Adattatore di stato
  • Stato della connessione di una scheda di rete locale (connessa / disconnessa)
Lasso di tempo
  • Alcuni giorni della settimana e il telaio di tempo (ad esempio, Sa + Do, 8:00-18:00)

Anche se non c'è alcuna limitazione tecnica per la quantità di condizioni per profilo cercare di mantenere il più semplice possibile. Nella maggior parte dei casi è assolutamente sufficiente includere uno o due condizioni per ottenere il risultato desiderato.

È possibile specificare che tutti o almeno una delle condizioni devono essere soddisfatte per attivare quel profilo.

Nota: I profili con ID più basse hanno una priorità più alta in condizioni caso per più profili sono soddisfatte contemporaneamente.

Per impostazione predefinita, il processo di attivazione è iniziato con un messaggio in basso a destra con un ritardo conto alla rovescia personalizzabile, che consente di opzionalmente interrompere il processo di attivazione. In alternativa, una notifica di Windows nativo può essere utilizzato o il profilo può essere attivato senza alcun messaggio in background. In quest'ultimo caso l'icona del vassoio sarà animato durante il processo di attivazione e cambia colore temporaneamente secondo il risultato di attivazione ti informa visivamente sul processo automatizzato.

Per risparmiare risorse di sistema NetSetMan non confronta le impostazioni correnti per le impostazioni del profilo da attivare. Questo significa che se si modifica qualsiasi impostazione manualmente, NetSetMan non noterà il cambiamento.

AutoSwitch ha alcune limitazioni in XP: Windows NLM non è disponibile, Cambiamenti nelle reti WiFi circostanti potrebbe essere riconosciuta con un ritardo notevole.

Note

È possibile creare brevi note per ogni profilo se avete bisogno di ricordare o indicare lo scopo di un profilo. Se una nota è stato creato un profilo di sua icona apparirà di colore nella barra dei menu. La nota è quindi visibile in modalità completa come un suggerimento dell'icona e in modo compatto in cima le impostazioni del profilo.

NSM Servizio / account utente con limitazioni

NetSetMan richiede privilegi di amministratore per poter modificare le impostazioni di rete e di sistema per voi. Normalmente l'esecuzione di un tale applicazione non è possibile per un account utente limitato. Le versioni più recenti di Windows anche limitare gli account amministratore con UAC (User Account Control), che mostra un messaggio ad ogni avvio del programma.

Per risolvere questo problema NetSetMan include un componente speciale chiamato NSM servizio che fornisce l'applicazione principale con i privilegi necessari. Il servizio viene installato automaticamente, ma può anche essere installato manualmente / disinstallato nelle preferenze (privilegi di sistema> Attiva / Disattiva).

Il servizio installato è visibile nella finestra di dialogo Servizi di Windows (services.msc) e può essere controllato anche lì. Inoltre, può essere installato / disinstallato o avviato / arrestato con riga di comando:
nsmservice.exe / install / silent
nsmservice.exe / uninstall / silent
nsmService net start
arresto nsmService net

Se si ottiene l'errore "Firma non valida", assicurarsi che il programma exe non è stato modificato e il criterio di gruppo Windows "Disattiva aggiornamento automatico certificati principali" è disattivata.

Se si esegue NetSetMan con un account utente limitato in cui il servizio NSM non è disponibile riceverai un messaggio che spiega il problema. In alternativa è possibile credenziali di amministratore di ingresso da un account locale che NetSetMan memorizzerà crittografato nel suo file di impostazioni e utilizzare per lanciarsi con i privilegi necessari. Si noti che in questo caso l'applicazione verrà eseguita nel contesto dell'utente le cui credenziali sono stati utilizzati.

Controllare la FAQ di leggere alcune note aggiuntive su questa funzione.

Di pre-accesso Profile Switching

Una caratteristica molto unica e potente è la possibilità di attivare i profili nella schermata di accesso. Questo permette la preparazione di corretti parametri di rete di cui avete bisogno per il vostro account, attivando un profilo NetSetMan prima ancora l'accesso.

Per utilizzare questa funzione NSM servizio deve essere installato e l'opzione "Profilo passaggio dalla schermata di accesso a Windows" (Preferenze> privilegi di sistema) deve essere abilitata.

Si noti che questa funzione consente solo di modificare le impostazioni che vengono condivise tra tutti gli utenti. Impostazioni specifiche dell'utente non possono essere modificati al di fuori di una sessione utente.

Amministrazione NSM

Se si imposta NetSetMan per essere utilizzato da qualcun altro, si potrebbe desiderare di specificare le azioni che l'altra persona è autorizzato a svolgere. Con l'Amministrazione NSM (Preferenze> Amministrazione) si può fare esattamente questo.

Quando si attiva questa modalità, dovrete inserire una password di 6 o più caratteri. Solo con questa password sarete in grado di modificare le impostazioni NSM Amministrazione o eseguire le azioni che si è scelto di essere assicurato (eseguito NetSetMan, attivare profili, modificare le impostazioni del profilo, le impostazioni del programma cambia).

È anche possibile bloccare alcuni profili quindi l'utente non può modificare, ma è ancora in grado di attivarli. Oppure si può generalmente limitare l'accesso a determinate impostazioni come "gruppo di lavoro e del dominio", consentendo all'utente di configurare e utilizzare tutte le altre impostazioni.

Un checksum della password sarà scritta nel Registro di sistema. Assicurarsi che un utente normale non ha privilegi di scrittura a HKLM \ Software \ NetSetMan (su x64: HKLM \ Software \ Wow6432Node \ NetSetMan). Se si dimentica la password, è necessario eliminare manualmente quella chiave. Se il file di impostazioni è crittografato nessuno senza la password sarà in grado di visualizzare o modificarlo.

Attivazione della riga di comando

In alcuni casi particolari si potrebbe desiderare di cambiare il vostro profilo senza utilizzare la GUI di NetSetMan: modificare i profili con le scorciatoie, attivare un profilo all'avvio, includerlo in uno script complesso, ecc

Sono disponibili i seguenti parametri:

-a # Attiva il profilo con il numero #
-as # Attiva il profilo in modalità silenziosa (finestra di attivazione si chiuderà automaticamente se non si sono verificati errori)
-h / -hide Runs NetSetMan minimizzato

-c / -Compact Esegue NetSetMan in modalità compatta

Ulteriori Pro Caratteristiche
Gruppi Profilo

Con NetSetMan Pro si è in grado di creare un numero illimitato di gruppi di profilo. Ogni gruppo contiene 6 profili.

Per aggiungere, eliminare, ridisporre o rinominare un gruppo di profili utilizzare la gestione dei profili nella barra dei menu.

Per navigare tra tutti i gruppi di profili disponibili scegliere qualsiasi gruppo dal campo a discesa nella barra dei menu.

Se si dispone di più di un gruppo, i profili verranno visualizzati raggruppati nel menu della barra.

Browser / Proxy

Con NetSetMan Pro è possibile gestire le impostazioni proxy di Windows / Internet Explorer, Firefox e Opera (<= v12). Chrome e versioni successive di Opera non supportano le proprie impostazioni proxy ma invece utilizzano automaticamente le impostazioni di sistema.

Tutte le impostazioni sono destinati a imitare le interfacce originali per consentire la stessa configurazione che è possibile anche nei browser direttamente. Inoltre è possibile modificare la home page e nel caso in cui si sta utilizzando profili in Firefox o Opera sarete anche in grado di selezionare il nome del profilo.

Si prega di notare che Firefox deve essere chiuso prima di modificare le impostazioni.
Se si dispone di più profili di Firefox e il loro profilo modifiche alle cartelle (ad esempio a causa di un nuovo sistema) avrete bisogno di ri-selezionare il profilo nella lista in modo che viene aggiornato nel file delle impostazioni.

Dominio

Per unirsi a un dominio immettere il suo nome, il nome utente e la tua password. Il tuo nome utente e la password verranno memorizzati criptati nel file INI.

Di solito è sufficiente inserire il nome utente pura. Tuttavia, se si ottiene un errore si può provare a inserire uno dei due seguenti standard come il tuo nome utente:
a) NETBIOS: DOMINIO \ utente
b) UPN: user@domain.local

In NT domini un utente può essere unico membro di esattamente un dominio allo stesso tempo. Ciò significa che, unendo un altro dominio o di un gruppo di lavoro la tua iscrizione nel dominio precedente viene persa fino a quando il sistema è rientrata.

NetSetMan si può partecipare solo a un dominio se l'account utente di dominio ha il permesso di farlo.
Questo privilegio account utente è chiamato "Aggiungi workstation al dominio" e deve essere abilitato nel tuo account. Se non si dispone di questo privilegio, rivolgersi al proprio amministratore di rete per garantire che.
